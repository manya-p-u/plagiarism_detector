Syllable Transliteration is a web-based application that converts English words into syllable-based transliterations across multiple languages such as Hindi, Marathi, Greek, Latin, and French.
It uses a Flask backend API for transliteration processing and a React frontend for an interactive user interface.
The system performs syllable segmentation, counts syllables, and retrieves transliteration mappings from a structured dataset with fallback syllabification logic.
This project demonstrates multilingual natural language processing concepts through a simple, user-friendly transliteration platform.
