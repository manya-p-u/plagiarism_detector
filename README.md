This project is a web-based plagiarism detection system that analyzes similarity between documents using Natural Language Processing (NLP) techniques. It supports multiple file formats such as PDF, DOCX, TXT, and CSV, extracts textual content, and computes similarity using TF-IDF vectorization with cosine similarity. The system highlights matching content, provides graphical visualization of plagiarism percentage, and generates downloadable reports. Built using Python, Flask, HTML, CSS, and Scikit-learn, it offers a simple and user-friendly interface for plagiarism analysis.
